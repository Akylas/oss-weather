# Notifications

Stay informed with weather alerts and notifications.

## Overview

OSS Weather can send notifications to keep you updated on weather conditions, severe weather, and daily briefings.

## Notification Types

### Weather Alerts

Severe weather notifications from providers:
- **Storm Warnings** - Thunderstorms, severe storms
- **Wind Advisories** - High winds, gales
- **Precipitation Alerts** - Heavy rain, snow
- **Temperature Extremes** - Heat waves, cold snaps
- **Other Hazards** - Fog, ice, flooding

### Daily Briefings

Scheduled notifications:
- **Morning Briefing** - Today's forecast
- **Evening Briefing** - Tomorrow's forecast
- **Custom Times** - Set your own schedule

### Condition Alerts

Custom notifications for:
- Temperature thresholds
- Precipitation probability
- Wind speed limits
- Air quality levels
- Custom conditions

### Location Alerts

Notifications for:
- Current location weather changes
- Saved locations
- Travel destinations

## Setting Up Notifications

### Enable Notifications

1. Open Settings
2. Go to Notifications
3. Enable notifications
4. Grant notification permission (if prompted)

### Configure Alert Types

Choose which alerts to receive:
- Severe weather (recommended)
- Daily briefings
- Temperature alerts
- Precipitation alerts
- Air quality alerts
- Custom conditions

### Set Schedule

For daily briefings:
- Morning time (e.g., 7:00 AM)
- Evening time (e.g., 8:00 PM)
- Custom schedules
- Weekend vs weekday

## Customizing Notifications

### Alert Thresholds

**Temperature**:
- Above/below specific value
- Heat index thresholds
- Wind chill limits

**Precipitation**:
- Probability threshold (e.g., >60%)
- Amount threshold (e.g., >10mm)
- Type (rain, snow, or both)

**Wind**:
- Speed threshold
- Gust threshold

**Air Quality**:
- AQI threshold
- Specific pollutants

### Notification Content

Customize what to include:
- Current conditions
- Hourly forecast
- Daily forecast
- Specific metrics
- Recommendations

### Quiet Hours

Set do-not-disturb periods:
- Sleep hours (no notifications)
- Work hours (limited notifications)
- Custom quiet periods

## Notification Behavior

### Priority Levels

**High Priority** (Severe Weather):
- Makes sound
- Vibrates
- Bypasses quiet hours
- Shows prominently

**Normal Priority** (Daily Briefings):
- Respects quiet hours
- Standard notification
- Can be customized

**Low Priority** (Information):
- Silent
- No vibration
- Minimal interruption

### Actions

Notifications can include:
- **View Details** - Open app to location
- **Dismiss** - Clear notification
- **Snooze** - Remind later (if applicable)

## Managing Notifications

### Notification Settings

**iOS**:
1. Go to Settings → Notifications
2. Find OSS Weather
3. Configure notification style
4. Set sounds and badges

**Android**:
1. Long press notification
2. Tap settings icon
3. Choose notification categories
4. Set importance levels

### Reducing Notifications

If too many:
- Disable less important types
- Increase thresholds
- Use quiet hours
- Limit to critical only

### Troubleshooting

**Not Receiving Notifications**:
1. Check app notification permission
2. Verify settings enabled
3. Check quiet hours
4. Check battery optimization
5. Ensure background updates enabled

**Too Many Notifications**:
1. Adjust thresholds
2. Disable non-critical alerts
3. Set quiet hours
4. Limit locations

## Battery Impact

Notifications require:
- Background app access
- Periodic weather checks
- Internet connectivity

Minimize battery impact:
- Reasonable check intervals
- Limit alert types
- Use efficient settings

## Privacy

Notifications:
- Generated locally on device
- Based on your weather data
- No data sent elsewhere
- No tracking

## Best Practices

Enable:
- Severe weather (safety)
